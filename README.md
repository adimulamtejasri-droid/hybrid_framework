 Hybrid Automation Framework

 Project Overview
This project is a **Hybrid Test Automation Framework** built using **Selenium WebDriver, Java, TestNG, and Maven**.  
It combines **Data-Driven, Keyword-Driven, and Page Object Model (POM)** approaches to provide a scalable and maintainable automation solution.



 🛠 Tech Stack
Java
Selenium WebDriver
TestNG
 Maven
WebDriverManager
